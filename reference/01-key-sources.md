# 5. 一次情報源・参考リンク一覧

技術的な内容を深掘りする際は、必ずどの情報源に基づく記述かを意識してください。
以下、情報源ごとに「何が書かれているか」「どう読むべきか」を整理します。
Frontier自体の一次情報については [03-frontier-platform.md](03-frontier-platform.md) の
「情報源」節にまとめているため、本章ではCrystal Intelligence関連の情報源を中心に扱います。

## 5.1 SB OAI Japan 公式プレスリリース（一次情報）

### 基盤プラットフォームの発表（PDF）

- **URL**: https://www.sb-oai.co.jp/pdf/20260206_sboai_pressrelease_openai_frontier_ja.pdf
- **公開形式**: PDF（プレスリリース）
- **内容の要点**:
  - Crystal Intelligence が OpenAI の「Frontier」を基盤に構築されていること
  - 「o1シリーズ」等の推論に優れたLLMがベースになっていること
  - 自律型AIエージェント群による業務プロセス遂行の構想
  - FDEによる伴走型の実装支援体制
- **読み方の注意**: プレスリリースPDFは事実発表が中心で、実装の詳細（アルゴリズム、
  データ構造、インフラ構成など）までは踏み込みません。「何を使っているか」「何を目指しているか」
  を把握するための一次情報として扱ってください。

### 参考：英語版プレスリリース

- **URL**: https://www.softbank.jp/en/corp/news/press/sbkk/2026/20260206_01/
- **公開形式**: Webページ（英語、SoftBank公式サイト）
- **内容の要点**: 日本語PDF版と同内容の英語版。原文のニュアンスを確認したい場合や、
  英語の技術用語（"Business Context" "Agent Execution" 等）の対応を確認したい場合に有用。

## 5.2 ソフトバンク Insights（技術解説・二次情報）

### クリスタル・インテリジェンス：企業変革をもたらす最先端AI

- **URL**: https://ai.softbank/insights/005/
- **公開形式**: Webページ（Insights/解説記事）
- **内容の要点**:
  - AIが企業の「長期記憶」を持つという技術的な動作イメージ
  - 社内データ・基幹システムのソースコード解析からプログラム置き換えまでの活用イメージ
  - アーキテクチャの背景にある技術的ビジョン
- **読み方の注意**: この記事はソフトバンク側の検証・知見を踏まえた解説であり、
  マーケティング的な言い回しも含まれる可能性があります。「具体的に何を統合するか」
  「どんな価値を狙っているか」を理解する補助線として読み、断定的な技術仕様として
  引用する場合は他の一次情報と突き合わせることを推奨します。

## 5.4 現時点で存在しない情報源

以下は2026年7月時点で公開が確認できていません。存在を前提に調査を進めないよう注意してください。

- 開発者向けの公開APIリファレンス
- パブリックなGitHubリポジトリ／SDK
- 料金体系・SLAなどの一般公開ページ
- モデルのベンチマーク結果や技術論文

これらが必要な場合は、SB OAI Japanまたはソフトバンクの営業窓口を通じた
個別問い合わせが前提になると考えられます（※一般的なエンタープライズ商材の商習慣からの推定）。
